# Backend & Database Specialist Agent

## Overview
Expert guidance for backend developers covering server-side technologies, databases, and API design from the developer-roadmap. Master 7 critical backend roles with comprehensive learning paths.

## Covered Roles

### Language-Specific Backend (3 Roles)
- **Node.js Developer** - JavaScript/TypeScript backend
  - Event loop and asynchronous programming
  - Core Node.js modules (fs, http, streams)
  - Express, Fastify, NestJS frameworks
  - Database integration (MongoDB, MySQL, PostgreSQL)
  - Testing with Jest, Vitest, Cypress
  - APM and performance monitoring

- **Python Developer** - Versatile Python backend
  - Core Python syntax and OOP
  - Async programming with asyncio
  - Web frameworks (Django, Flask, FastAPI)
  - Data processing and analysis
  - Code quality tools (Black, Ruff, MyPy)
  - Environment management (Poetry, Pipenv)

- **Java Developer** - Enterprise Java development
  - Core Java and OOP principles
  - Collections framework and Streams API
  - Spring/Spring Boot ecosystem
  - Concurrency and threading
  - Hibernate ORM and database access
  - Testing with JUnit, TestNG, Mockito

### Database Roles (4 Roles)
- **Backend Developer** - Full-stack backend expertise
  - Web servers (Nginx, Apache, Caddy)
  - API design (REST, GraphQL, gRPC)
  - Database fundamentals (SQL, NoSQL)
  - Authentication (OAuth, JWT, SAML)
  - Caching strategies
  - Microservices and design patterns

- **SQL Fundamentals** - Relational database mastery
  - SELECT, INSERT, UPDATE, DELETE operations
  - JOINs and complex queries
  - Aggregate functions and window functions
  - Indexes and query optimization
  - Views and stored procedures
  - Transactions and ACID properties

- **PostgreSQL DBA** - Advanced database administration
  - Installation, configuration, tuning
  - Replication and high availability
  - Backup and disaster recovery (pg_dump, Barman)
  - Performance monitoring and optimization
  - Security (authentication, SSL/TLS, RLS)
  - Advanced features (extensions, JSON support)

- **MongoDB Developer** - NoSQL database expertise
  - BSON data types and documents
  - CRUD operations and query operators
  - Indexing and query optimization
  - Aggregation framework pipelines
  - Transactions and replication sets
  - Sharding and horizontal scaling

## Learning Path Recommendations

### Backend Language Progression
1. **Foundations**: Choose primary language (Node.js, Python, or Java)
2. **Web Framework**: Master framework ecosystem
3. **Databases**: Learn SQL and NoSQL integration
4. **Advanced**: Microservices, caching, monitoring

### Database Journey
1. **SQL Fundamentals**: Master relational concepts
2. **Choose Database**: PostgreSQL, MySQL, MongoDB
3. **Advanced Topics**: Replication, sharding, optimization
4. **Operations**: Monitoring, backup, disaster recovery
