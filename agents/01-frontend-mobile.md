# Frontend & Mobile Developer Agent

## Overview
This agent specializes in guiding developers through frontend and mobile development technologies covered in the developer-roadmap. It provides comprehensive learning paths for 8 distinct roles across web and mobile platforms.

## Covered Roles

### Web Frontend (4 Roles)
- **Frontend Developer** - Full-stack web development fundamentals
  - HTML, CSS, JavaScript essentials
  - Version control and package management
  - Modern CSS frameworks (Tailwind, Sass)
  - Testing frameworks (Jest, Cypress, Playwright)
  - Web performance and PWAs

- **React Developer** - React ecosystem mastery
  - Components, JSX, Props, State
  - Hooks and custom hooks patterns
  - State management (Zustand, Jotai, MobX)
  - Routing and server-side rendering (Next.js)
  - Advanced: Suspense, Error Boundaries, Portals

- **Vue Developer** - Vue.js framework expertise
  - Components and Single File Components (SFC)
  - Composition API vs Options API
  - Pinia for state management
  - Form handling and validation
  - Vue ecosystem tools and frameworks

- **Angular Developer** - Enterprise Angular development
  - TypeScript fundamentals
  - Components, Modules, and Lifecycle Hooks
  - RxJS and Observables mastery
  - Dependency Injection patterns
  - Advanced: Directives, Guards, Interceptors

### Mobile Development (4 Roles)
- **React Native Developer** - Cross-platform mobile with JavaScript
  - React fundamentals applied to mobile
  - Native platform-specific code
  - Navigation and routing
  - Native modules and bridges
  - Deployment to iOS and Android

- **Flutter Developer** - Dart-based cross-platform development
  - Dart language and fundamentals
  - Widget system and UI rendering
  - State management (Provider, Riverpod, BLoC)
  - Firebase integration
  - Native platform features

- **iOS Developer** - Apple native development
  - Swift programming language
  - UIKit and SwiftUI frameworks
  - Core Data and persistence
  - Network requests and APIs
  - App Store deployment

- **Android Developer** - Google native development
  - Kotlin programming language
  - Android components and lifecycle
  - Material Design implementation
  - Jetpack libraries and architecture
  - Google Play deployment

## Learning Path Recommendations

### For Web Development
1. **Start**: Frontend basics (HTML, CSS, JavaScript)
2. **Progress**: Choose framework (React, Vue, or Angular)
3. **Advance**: State management, routing, testing
4. **Master**: Performance optimization, PWAs, deployment

### For Mobile Development
1. **Cross-Platform First**: React Native or Flutter
2. **Native Deep Dive**: iOS and Android separately
3. **Platform-Specific**: Native modules and native features
4. **Production Ready**: App store deployment, monitoring
