# /explore-roles - Browse All Developer Roles

Explore and compare all 65+ developer roles from the roadmap. Find the perfect fit for your career.

## Usage

```
/explore-roles [filter]
```

## Examples

- `/explore-roles` - See all roles
- `/explore-roles frontend` - Show frontend-related roles
- `/explore-roles backend` - Show backend-related roles
- `/explore-roles devops` - Show DevOps roles
- `/explore-roles data-science` - Show data science roles

## Features

- **Detailed Descriptions**: What each role involves
- **Skills Required**: Key skills for each role
- **Learning Paths**: How to prepare for each role
- **Career Progression**: How roles relate to each other
- **Salary Ranges**: Typical compensation (where available)
- **Job Market**: Demand and opportunities

## Comparison

Compare multiple roles side-by-side:
```
/explore-roles compare frontend react angular
```

This shows:
- Skills overlap
- Learning time estimates
- Career paths
- Salary ranges
- Job availability

## Filtering Options

- **By Career Stage**: Entry-level, Mid-level, Senior, Lead
- **By Technology**: JavaScript, Python, Go, Kubernetes, AWS, etc.
- **By Specialization**: Web, Mobile, DevOps, AI/ML, Security
- **By Team Size**: Solo, Startup, Scale-up, Enterprise
- **By Remote-Friendly**: Full remote, Hybrid, On-site

## Tips

1. **Start Broad**: Explore categories first
2. **Read Descriptions**: Understand what each role does
3. **Check Skills**: Identify which skills interest you
4. **Compare Paths**: See how roles connect
5. **Research Market**: Understand job availability
