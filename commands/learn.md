# /learn - Start Learning Path

Start a guided learning journey for any developer role from the 65+ roles in the roadmap.

## Usage

```
/learn [role-name]
```

## Examples

- `/learn react` - Learn React development
- `/learn backend-python` - Learn Python backend
- `/learn devops-kubernetes` - Learn Kubernetes
- `/learn machine-learning` - Learn ML engineering
- `/learn product-management` - Learn product management
- `/learn security` - Learn cybersecurity

## What You'll Get

1. **Structured Learning Path** - Step-by-step progression from basics to advanced
2. **Key Topics** - All essential topics for the role
3. **Recommended Resources** - Official docs, courses, tutorials
4. **Practice Projects** - Real-world projects to build
5. **Time Estimates** - Expected learning duration for each phase
6. **Assessment Checkpoints** - Verify your progress

## Features

- **Personalized Path**: Recommends learning order based on your current level
- **Interactive Progress**: Track your progress through the learning path
- **Resource Links**: Direct links to official documentation and tutorials
- **Skill Building**: Specific skills to develop for each role
- **Projects**: Real-world projects to apply your knowledge
- **Community**: Connect with others learning the same role

## Tips

1. **Start Small**: Begin with fundamentals before advanced topics
2. **Practice Projects**: Build real projects to reinforce learning
3. **Join Community**: Engage with others learning the same path
4. **Review Regularly**: Revisit earlier topics to strengthen foundations
5. **Take Breaks**: Learning is a marathon, not a sprint
