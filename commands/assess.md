# /assess - Knowledge Assessment

Assess your current knowledge level and get personalized recommendations for development.

## Usage

```
/assess [role]
/assess [skill-area]
```

## Examples

- `/assess` - General assessment across all areas
- `/assess react` - Assess React knowledge
- `/assess backend` - Assess backend development skills
- `/assess devops` - Assess DevOps expertise
- `/assess machine-learning` - Assess ML knowledge

## What You Get

### 1. Skill Assessment Report
```
Frontend Development
├─ HTML/CSS: Intermediate ⭐⭐⭐
├─ JavaScript: Advanced ⭐⭐⭐⭐
├─ React: Intermediate ⭐⭐⭐
├─ State Management: Beginner ⭐⭐
└─ Testing: Beginner ⭐⭐
```

### 2. Proficiency Levels
- **Beginner** (0-3 months): Basic understanding
- **Intermediate** (3-12 months): Comfortable with most tasks
- **Advanced** (12+ months): Expertise, can mentor others
- **Expert** (2+ years): Deep knowledge, innovation

### 3. Strengths & Weaknesses
- **Strengths**: Areas where you excel
- **Gaps**: Areas needing improvement
- **Opportunities**: High-impact skills to develop

### 4. Personalized Roadmap
```
Next 3 months:
1. Master State Management (React)
2. Learn Advanced Testing (Cypress, Playwright)
3. Performance Optimization fundamentals

Next 6 months:
1. Full-stack development
2. Advanced TypeScript patterns
3. GraphQL implementation

Next 12 months:
1. System design
2. Microservices architecture
3. Technical leadership
```

### 5. Resource Recommendations
- Courses and tutorials matching your level
- Books and articles for deep learning
- Projects to practice specific skills
- Communities and mentors

## Assessment Methods

### Quick Assessment (5-10 minutes)
- Quick questions about your experience
- Self-evaluation of skills
- Instant results and recommendations

### Deep Assessment (30-45 minutes)
- Comprehensive skill evaluation
- Problem-solving scenarios
- Technical knowledge questions
- Experience review
- Detailed report with actionable insights

### Project-Based Assessment
- Build a small project
- Show your code
- Get detailed feedback
- Identify growth areas

## Growth Recommendations

Based on your assessment, you'll get:

1. **Skill Development Plan** - What to learn next
2. **Time Estimates** - How long it will take
3. **Resource List** - Where to learn
4. **Practice Projects** - What to build
5. **Milestone Goals** - Track your progress

## Tips

1. **Be Honest**: Accurate assessments help more
2. **Reassess Regularly**: Track growth over time
3. **Focus on Gaps**: Develop weak areas
4. **Build Projects**: Apply learning immediately
5. **Join Communities**: Learn from others

## Follow-Up

After assessment, you can:
- `/learn [recommended-role]` - Start a learning path
- `/recommend` - Get career recommendations
- `/explore-roles` - Compare with other roles
